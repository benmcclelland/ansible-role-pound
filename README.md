# Ansible Role: Pound

Installs Pound on RedHat/CentOS or Debian/Ubuntu.

## Requirements

None.

## Dependencies

None.

## Example Playbook

    - hosts: load-balancer
      sudo: yes
      roles:
        - { role: ansible-role-pound }
